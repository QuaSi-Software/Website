Contact us

Don't hesitate to contact us for general questions on the project and related organisations.

Q&A for technical questions

Several of the developers of the various QuaSi tools are active on the openmod forum. If you need help with using one of the tools, you can ask in the Q&A section of the forum and tag the post correspondingly. In all likelihood it won't take long for some of the developers to notice and reply.

Issues, bugs and feature requests

For bugs and specific issues you can visit the repositories of the tools and create an issue there. This is also the place to request that a feature you need might be added. Unfortunately we cannot guarrantee that it will be possible to implement, but it doesn't hurt to ask.
